ImageViewer
===========

图片查看器，支持多点触控等

功能支持：

1.多点触控，拖拽，伸缩。

2.边界范围检查，内容边界检查。(网上很多代码没有支持边界检查，有的支持了带不支持黑边检查，例如图片很宽，imageview Fitcenter scaletype 会在height上补全透明边，如果放大后黑边会变大如果不检查内容边界只做view边界检查会发生屏幕显示都是黑色多余得边)

3.放大图片最大最小倍数检查。

4.Zooming a view 动画

5.双击放大缩小

6.支持click事件
